Große Anleitung (alles erklärt)

1) Was ist das hier?

Das ist eine mobile Lern-Webseite für deine Englisch-Klausur (Q1/NRW). Du lernst:

  • Vokabeln (200) passend zu Utopia/Dystopia/Technology/Genetics/Climate/Society
  • Advanced Vocabulary & Transitions (klausurtaugliche Verknüpfungen, Academic Verbs, Hedging)
  • Sentence Starters (Analyse/Argumentation/Mediation)
  • Textsorten & Cheat Sheets (Non-fiction types, Strukturen, dystopian features, Mediation-Regeln)

2) Lernen (Flip) – so benutzt du’s

  • Rubrik wählen → Lernen (Flip)
  • Vorderseite lesen → in deinem Kopf antworten → Flip → vergleichen
  • ✅ = „kann ich“ (zählt als richtig, entfernt ggf. „schwer“)
  • ❌ = „schwer“ (markiert Karte als Schwäche)
  • Schwächen trainieren: Nur schwer aktivieren

3) Quiz (Eingabe) – so wird’s klausurtauglich

  • Rubrik wählen → Quiz (Eingabe)
  • Du tippst die Kernantwort (z.B. „surveillance“, „as a result“, „to warn against“)
  • Die Prüfung ist tolerant – du musst nicht wortwörtlich treffen

4) Master-Lernzettel Download (TXT) + PDF mit ChatGPT

Sobald du genug geübt hast (Freischaltung), kannst du einen Master-Lernzettel als Textdatei downloaden. Der enthält alles (Vokabeln + Advanced + Starters + Textsorten) und ganz oben einen fertigen Prompt.

So machst du daraus eine PDF:
1) Download öffnen → alles kopieren
2) In ChatGPT (oder andere KI) einfügen
3) KI erstellt dir eine schön formatierte PDF (mit Tabellen/Boxen/Überschriften)

5) Fertiger Prompt (steht auch im Download)

Erstelle aus dem folgenden Lernzettel eine professionell formatierte PDF zum Lernen für eine Englisch-Q1-Klausur (NRW). Anforderungen: - klare Gliederung mit Überschriften & Inhaltsverzeichnis (wenn möglich) - Vokabeln als Tabellen: Deutsch | Englisch | Synonyme/Chunks (falls vorhanden) - Advanced Vocabulary & Transitions in farbigen Boxen (Kategorien: contrast, cause/effect, addition, hedging, evaluation, academic verbs) - Sentence Starters nach Funktionen sortieren (introducing arguments, contrast, conclusion, analysis, intention, mediation) - Textsorten & Merkmale als übersichtliche Checklisten (Non-fiction types, structures, dystopia/utopia features, mediation rules) - 1–2 Beispiel-Absätze, die zeigen, wie man Advanced Vocabulary + Starters in einer Analyse verwendet - Layout: modern, sauber, schülerfreundlich; gut druckbar (A4) und gut auf Handy/Tablet lesbar Ausgabe: PDF (oder druckfertiges Layout + klare PDF-Export-Anleitung). 6) GitHub Pages (damit Freunde es öffnen können)

  • Datei muss im Repo index.html heißen
  • Repo → Settings → Pages → Deploy from branch → main / root
  • Dann Link teilen: https://username.github.io/repo/

  • Öffne es über GitHub Pages (nicht als lokale Datei im Preview)
